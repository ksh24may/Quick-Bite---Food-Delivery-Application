# Quick-Bite---Food-Delivery-Application
 A native Android application using Android SDK and XML layouts, implementing RecyclerView and CardView for menu and cart display, Intents for navigation between screens, Toast messages for user feedback, and basic UI animations for an enhanced user experience.
 
 📱 Features

- 🧾 **Menu Display**
  - Uses RecyclerView and CardView for attractive UI
- 🛒 **Cart System**
  - Add and manage food items easily
- 🔄 **Navigation**
  - Implemented using Intents between multiple screens
- 🔔 **User Feedback**
  - Toast messages for actions and alerts
- 🎨 **UI Enhancements**
  - Basic animations for better user experience


## 🛠️ Technologies Used

- **Java / Kotlin**
- **Android SDK**
- **XML (UI Design)**
- **RecyclerView & CardView**
- **Intents & Activity Lifecycle**
